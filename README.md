# hermes-mcp-bridge
A zero-friction stdio MCP bridge connecting Cursor Desktop to a local Hermes Agent — no OAuth, no tunnels, no Dynamic Client Registration. Preserves session context across REPL turns via X-Hermes-Session-Id. 零摩擦 stdio MCP 桥接器 — Cursor Desktop 直连本地 Hermes Agent，无需 OAuth、隧道或 Dynamic Client Registration，通过 X-Hermes-Session-Id 保持 REPL 跨轮次上下文。
